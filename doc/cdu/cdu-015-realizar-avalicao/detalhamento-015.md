# CDU015. Realizar avaliação

- **Ator principal**: Leitor
- **Atores secundários**:	 
- **Resumo**: Leitor poderá avaliar um livro após terminar de lê-lo.
- **Pré-condição**: O leitor deve estar autenticado no sistema e ter finalizado a leitura de um livro.
- **Pós-Condição**: A avaliação do livro é registrada no sistema.

## Fluxo Principal
1. &emsp;O leitor acessa a página do livro que acabou de ler.
2. &emsp;O leitor clica no botão para avaliar o livro.
3. &emsp;O sistema exibe um formulário de avaliação, permitindo que o leitor forneça uma nota e, opcionalmente, um comentário sobre o livro.
4. &emsp;O leitor preenche o formulário de avaliação com a nota e, opcionalmente, um comentário.
5. &emsp;O leitor submete o formulário de avaliação.
6. &emsp;O sistema valida as informações fornecidas:
   - &emsp;Se as informações estiverem corretas, o sistema registra a avaliação do livro.
7. &emsp;O sistema confirma que a avaliação foi registrada com sucesso e atualiza a média de avaliações do livro, se aplicável.

> Obs. as seções a seguir apenas serão utilizadas na segunda unidade do PDSWeb (segundo orientações do gerente do projeto).
