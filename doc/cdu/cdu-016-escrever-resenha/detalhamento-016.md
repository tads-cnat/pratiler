# CDU016. Escrever resenha

- **Ator principal**: Leitor
- **Atores secundários**:	 
- **Resumo**: Leitor poderá escrever uma resenha sobre um livro após terminar de lê-lo.
- **Pré-condição**: O leitor deve estar autenticado no sistema e ter finalizado a leitura de um livro.
- **Pós-Condição**: A resenha do livro é registrada no sistema.

## Fluxo Principal
1. &emsp;O leitor acessa a página do livro que acabou de ler.
2. &emsp;O leitor clica no botão para escrever uma resenha.
3. &emsp;O sistema exibe um formulário de resenha, permitindo que o leitor forneça o texto da resenha.
4. &emsp;O leitor preenche o formulário de resenha com o texto.
5. &emsp;O leitor submete o formulário de resenha.
6. &emsp;O sistema valida as informações fornecidas:
   - &emsp;Se as informações estiverem corretas, o sistema registra a resenha do livro.
7. &emsp;O sistema confirma que a resenha foi registrada com sucesso e a exibe na página do livro.

> Obs. as seções a seguir apenas serão utilizadas na segunda unidade do PDSWeb (segundo orientações do gerente do projeto).
