# CDU020. Ver feed dos seguindo

- **Ator principal**: Leitor
- **Atores secundários**:	 
- **Resumo**: O leitor verá as postagens dos leitores que ele segue
- **Pré-condição**: Ser usuário do tipo leitor, estar logado no sistema e existir comentários
- **Pós-Condição**: O leitor terá acesso as postagens dos leitores que ele segue

## Fluxo Principal
&emsp;&emsp;Na tela principal, o leitor pode acessar a página com as postagens dos leitores que ele segue, acessando a opção "seguindo" na barra de navegação acima.<br>
&emsp;&emsp;Selecionando essa opção, o leitor terá acesso a uma listagem de postagens dos leitores que segue, ordenadas por data de publicação, da mais recente para a mais antiga.

## Fluxo de Exceção I - Não há postagens
&emsp;&emsp;Ao selecionar a página de postagens de "seguindo", se os leitores que o leitor segue não tiverem postagens, o sistema informará que não tem postagens nesta página.
