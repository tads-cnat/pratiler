# Manual da Desenvolvedor

[Orientações para os desenvolvedores do projeto](doc/guia-ds/guia.md)

Lista dos Casos de Uso:

* Adicionar Livro à Estante: Os usuários podem adicionar livros à sua estante pessoal para acompanhar sua leitura.

* Buscar Livros: Os usuários podem pesquisar livros por título, autor ou categoria para encontrar novas leituras.

* Avaliar Livro: Os usuários podem avaliar os livros que leram, atribuindo uma classificação e deixando comentários.

* Ver Perfil do Usuário: Os usuários podem visualizar o perfil de outros usuários para ver seus livros favoritos, avaliações e atividades recentes.

* Seguir Usuário: Os usuários podem seguir outros usuários para receber atualizações sobre suas atividades de leitura.

* Recomendar Livro: Os usuários podem recomendar livros para outros usuários com base em seus interesses e preferências de leitura.

* Ver Lista de Livros Recomendados: Os usuários podem visualizar uma lista de livros recomendados com base em suas preferências de leitura e nas recomendações de outros usuários.

* Marcar Livro como Lido: Os usuários podem marcar livros como "lidos" em sua estante pessoal para acompanhar seu progresso de leitura.

* Configurações de Privacidade: Os usuários podem configurar suas preferências de privacidade, como quem pode ver suas avaliações e atividades de leitura.
